Es posible que el módem también tenga un modo de prueba automática (consulte la documentación del módem). En este modo, la computadora manda de forma repetida un patrón de datos al modem y éste los regresa a la computadora. Esta prueba de bucle invertida interna verifica que el módem funciona correctamente y que el envío y la recepción de datos realmente se esté realizando de forma correcta. Puede utilizar este modo si los pasos anteriores no lograron establecer comunicación. Si esta prueba falla, esto indica un problema con el módem. Consulte la documentación del módem para ver cómo proceder. Una portadora es un tono que el módem espera escuchar antes de poder enviar datos. Es como decir "Hola" después de que suena el teléfono y has descolgado el auricular. Después de decir "Hola", se puede establecer la comunicación y se puede transmitir información de la persona que llama. Esta portadora se puede escuchar cuando se usa un módem inteligente que marca el teléfono automáticamente. Primero escucharía sonar el teléfono a través del altavoz del módem y luego sonaría un tono agudo. Esta señal audible es la portadora y le dice al sistema que el módem está listo para aceptar datos, ya sea de su sistema o del otro sistema. La luz del indicador CD (Carrier Detected o Portadora detectada) está iluminada. Esto indica que el otro sistema ha respondido y se le ha ha detectado la señal portadora. Si no se recibe la señal portadora, podría indicar un número incorrecto, una señal de línea ocupada o un problema en el otro extremo.
