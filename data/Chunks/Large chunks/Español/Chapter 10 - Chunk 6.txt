Leer el manual del dispositivo. Lea la documentación del dispositivo tan pronto como obtenga el nuevo periférico. Esto lo familiarizará con el nuevo dispositivo, le indicará cómo usarlo de manera más eficiente y le mostrará qué información está disponible y dónde se puede encontrar. Específicamente, el manual proporciona instrucciones sobre la instalación, configuración, uso, operación, mantenimiento y solución de problemas adecuados del dispositivo. La documentación cubre primero las instrucciones de desembalaje e instalación, que deben seguirse cuidadosamente para minimizar posibles problemas. Luego, el manual describe el funcionamiento, las diversas funciones y características especiales, y ofrece ejemplos de aplicaciones y capacidades. Según sea necesario, se proporcionan procedimientos para diversas opciones y configuraciones. El manual también cubre la integración del dispositivo con otros componentes y software del sistema. Se proporcionan sugerencias sobre la configuración de hardware y software existente para que el nuevo dispositivo funcione de manera adecuada y eficiente. Si fuera el caso, la documentación también describe cómo instalar el software adjunto y cómo programar el dispositivo. El manual generalmente incluye una guía de solución de problemas que describe los problemas que pueden ocurrir y sugiere procedimientos de solución a seguir en respuesta a estos problemas.
