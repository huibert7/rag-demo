Si el módulo de diagnóstico de comunicación en serie falla, vuelva a cargar el programa de comunicación y asegúrese de que los parámetros estén configurados correctamente. ¿Está configurada correctamente la velocidad en baudios? ¿Se especificó el tamaño de palabra o carácter apropiado? ¿El otro sistema espera que se habilite un bit de paridad? De ser posible, verifique el dispositivo con el que intenta comunicarse. Si es otro Apple II, ejecute el programa de diagnóstico para ver si su interfaz de comunicación funciona correctamente. Si se trata de un servicio en línea, llame a la línea de ayuda del servicio para obtener más asistencia técnica. Consulte el manual técnico de la interfaz o del sistema para ver si el cable de interfaz tiene algún requisito especial. RS-232 es un estándar relativamente flexible. Algunos fabricantes de cables utilizan un esquema diferente para las clavijas de sus cables RS-232. Estos cables pueden esperar datos de diferentes pines, esperar diferentes señales o tener señales unidas. Si este es el caso, modifique el cable para que coincida con la configuración de pines requerida o compre otro que esté diseñado especialmente para su sistema. Consulte al fabricante del cable para obtener más información. La mayoría de los módems brindan asistencia adicional para la resolución de problemas a través de su hilera de luces LED rojas en el panel de estado frontal (Fig. 8-7). Si su módem incluye dicho panel, verifique que las señales sean las esperadas. Por ejemplo, cuando el módem está correctamente conectado al Apple II y el software de comunicaciones se está ejecutando, las luces correspondientes a TR (Terminal Ready) y DSR (Data Set Ready) deben estar encendidas. Estas luces indican que la computadora y el módem están listos y que las señales de la interfaz están encendidas y listas para comunicarse entre sí. Si estas luces indicadoras no están encendidas, puede haber un problema con la conexión del cable de interfaz o con el módem. Cuando se envían datos, las luces indicadoras sobre RD (Recibir datos) o TD (Transmitir datos) están encendidas. Cuando el software de comunicaciones manda datos a través el módem, estas luces se iluminan para mostrar que los datos se están enviando.
