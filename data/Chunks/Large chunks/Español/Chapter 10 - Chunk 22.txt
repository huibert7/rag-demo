Cuando se encuentre un error, vea si se puede repetir. Luego pruebe otras variaciones de esta acción para ver si produce el mismo resultado. Esto le proporcionará más información sobre cómo solucionar el error. Tome notas en la lista del programa, como las líneas donde cree que podría existir el error y las posibles soluciones. Una vez que tenga suficiente información sobre el error, corrija el código editando las líneas afectadas. El error puede no ser obvio por lo que puede que sea necesario realizar investigaciones adicionales para descubrir cuál es la causa raíz del problema y así poder determinar posibles soluciones. Cuando crea que el error se ha solucionado, vuelva a probar el programa. Siga la misma secuencia de eventos que llevaron a que ocurriera el error y vea si vuelve a ocurrir. Pruebe todos los diferentes valores y combinaciones imaginables. Cuando finalmente esté satisfecho de que el error se solucionó en este lugar en particular, pruebe también todas las demás funciones con diferentes valores y combinaciones. A menudo, corregir un error creará otro error. No pierda la paciencia.
