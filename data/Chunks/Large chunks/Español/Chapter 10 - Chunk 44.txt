Integración con el Programa de diagnóstico del sistema. Una vez que el código para el diagnóstico de Game Paddle esté escrito y probado exhaustivamente por sí solo, debe integrarlo con el resto del programa de diagnóstico del sistema. Primero modifique la rutina del Menú principal del Programa de diagnóstico del sistema, que muestra y accede a todas las opciones de diagnóstico. Examine las líneas 50 a 90 del programa donde reside el módulo del Menú principal. Agregue el identificador de diagnóstico de Game Paddle a la declaración `PRINT` que enumera las opciones del menú en la pantalla. Además, inserte la instrucción `IF` apropiada para el identificador de diagnóstico de Game Paddle en las líneas 110-180, de modo que el programa se ramifique al principio del módulo de diagnóstico de Game Paddle cuando se presione la tecla apropiada, tal vez una G. Una vez hecho esto, agregue el código del módulo al programa. Un buen lugar para agregar el nuevo código es comenzando en la línea 6000, como se hizo en el ejemplo de este capítulo. Esto inicia el nuevo módulo donde se acaba el último módulo de diagnóstico.
