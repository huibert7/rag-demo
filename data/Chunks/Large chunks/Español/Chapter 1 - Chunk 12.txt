

| Herramientas requeridas                           | Uso                                                          |
| ------------------------------------------------- | ------------------------------------------------------------ |
| Destornilladores de punta plana                   | Para apretar y aflojar tornillos ranurados.                  |
| Destornilladores Phillips                         | Para apretar y aflojar los tornillos con ranuras cruzadas que se utilizan para sujetar los pernos. |
| Pinzas de punta de aguja                          | Para agarrar y/o doblar objetos pequeños como una extensión o una "mano amiga". |
| Alicates                                          | Para agarrar y/o doblar objetos pequeños, para sujetar tuercas más grandes. |
| Pelacables                                        | Para quitar el aislamiento plástico de los cables.           |
| Cortador de alambre                               | Para cortar alambre.                                         |
| Destornillador no conductor "varita de afinación" | Para apretar y aflojar tornillos en un entorno con carga eléctrica. |
| Extractor de teclas                               | To sacar teclas del teclado.                                 |
| Extractor de chips                                | Para quitar un chip IC insertado en su zócalo.               |
| Lima plana metálica                               | Para alisar o pulir una superficie de plástico como la tapa de una tecla. |

| Productos de limpieza                                    | Uso                                                          |
| -------------------------------------------------------- | ------------------------------------------------------------ |
| Aire comprimido con boquilla y extensión.                | Aire a presión utilizado para eliminar el polvo y la suciedad. |
| Freón o limpiador de contactos con boquilla y extensión. | Solución utilizada para limpiar contactos e interruptores en componentes de tarjetas de circuito impreso. |
| Alcohol isopropílico desnaturalizado.                    | Para limpieza general de hardware electrónico, cabezales de unidades de disquete y cabezas de impresión. |
| Cotonetes                                                | Para limpiar hardware y componentes en lugares de difícil acceso; utilizado con alcohol isopropílico. |
| Esponjas y papel o paño sin pelusa                       | Para limpiar componentes de hardware con alcohol isopropílico. |
| Papel de lija o borrador de lápiz                        | Para limpiar la oxidación de conectores y clavijas.          |
