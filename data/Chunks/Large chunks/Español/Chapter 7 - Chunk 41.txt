La primera pasada escribe una serie de registros de 254 bytes que contienen una cadena de 254 caracteres con valor CHR$ 64. El valor CHR$ 64 es un carácter que tiene una cadena de bits de 1000000. El diagnóstico escribe 239 registros con valor CHR$ 64 para llenar el disco. Mientras los registros se escriben en el disco, se muestra el mensaje "WRITING HIGH VALUES TO DISK", que en español significa "ESCRIBIENDO VALORES ALTOS EN EL DISCO". Se necesitan unos cinco minutos para que el disco se llene con estos registros. Una vez hecho esto, se cierra el archivo. El diagnóstico comienza inmediatamente con el segundo paso por el disco, que consiste en reabrir el archivo y releer los datos que se escribieron durante el primer paso. Los datos leídos se comparan con el valor 64 de CHR$. Mientras se leen y comparan los registros, se muestra el mensaje "READING HIGH VALUES FROM DISK", que en español se traduciría por "LEYENDO VALORES ALTOS DEL DISCO". Se necesitan unos cinco minutos para leer y comparar todos los registros.
