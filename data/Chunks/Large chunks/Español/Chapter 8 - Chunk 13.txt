Software de comunicaciones. Para utilizar la interfaz de comunicación serie, se necesita un software de comunicación que funcione desde la interfaz de comunicación serie. Un ejemplo de este tipo de software es un programa de comunicación por módem. La mayoría de los módems incluyen un software de comunicación. Lo mismo aplica para otros dispositivos de comunicación en serie que pueda adquirir. Algunas tarjetas de comunicación serie disponibles no requieren software independiente. Los interruptores DIP le permiten configurar los parámetros de comunicación y los programas especiales integrados en las ROM de la placa proporcionan conjuntos de comandos de comunicación para realizar todas las funciones necesarias del software de comunicación.
