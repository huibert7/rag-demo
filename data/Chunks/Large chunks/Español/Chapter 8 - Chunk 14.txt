Parámetros de comunicación. Dado que los datos se envían un bit a la vez en la comunicación en serie, se deben especificar ciertos parámetros al configurar por primera vez el software de comunicación. Estos parámetros pertenecen a la velocidad de transmisión de datos, el tamaño de los caracteres de datos que se envían, si se emplean bits de inicio y bits de parada y si se debe realizar una verificación de errores de paridad. La configuración de comunicación sirve para coordinar la comunicación y garantizar que la transferencia de datos sea exitosa y precisa. Los valores particulares especificados en estos parámetros dependen del dispositivo con el que se comunica la computadora. La documentación del módem describe cómo configurar los parámetros. La documentación del usuario del servicio en línea detalla los parámetros específicos necesarios para comunicarse con sus computadoras. La impresora, módem u otro dispositivo con el que el Apple II se va a comunicar debe tener las variables de la interfaz de comunicación configuradas de manera idéntica en su extremo para que este enlace de comunicación pueda establecerse y mantenerse. Estos parámetros se configuran a través del software o con interruptores en la placa de interfaz de comunicación. Consulte la documentación del dispositivo para obtener detalles específicos. Cuando ejecute el software de comunicación por primera vez, acceda al modo de configuración para definir estos parámetros de comunicación. Una vez configurados, estas opciones generalmente se almacenan en el disco del programa para que se implementen automáticamente cada vez que se utiliza el programa.
