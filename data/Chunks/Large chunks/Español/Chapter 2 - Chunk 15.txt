Ingresando el programa. Si no ha comprado el disquete, ingrese el código del programa como se indica en los Apéndices. Para hacer esto Encienda el sistema Apple. Applesoft BASIC se invoca automáticamente . Ingrese `NEW` para indicarle a BASIC que este es un programa nuevo. Ingrese el código fuente del Programa de diagnóstico del sistema exactamente como se indica en el Apéndice A. Guarde el programa después de cada cien líneas para protegerlo contra la pérdida de datos en caso de una falla de energía o del sistema. Utilice el comando `SAVE DIAG`. Esa parte del programa se guarda en el disquete. Nota: No hay nada especial en el nombre del programa "DIAG". Nómbralo como quieras. Sólo asegúrese de que sea un nombre descriptivo y que no haya ningún otro programa con ese nombre en ese disquete. Cuando haya ingresado y guardado todo el programa de diagnóstico del sistema en el disquete, ingrese `RUN DIAG` y pruebe cada función del programa para asegurarse de que no haya errores de sintaxis. Dichos errores indican errores tipográficos cometidos al ingresar al programa.
