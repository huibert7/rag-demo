Paridad. La paridad es un esquema de corrección de errores que se utiliza para garantizar una transmisión de datos precisa. Con paridad, el último bit de una palabra de datos se establece de modo que la suma del número de bits sea un número impar, para paridad impar, o un número par, para paridad par. Por ejemplo, si el sistema está configurado para paridad impar desde el módem a la computadora y se recibe un carácter o palabra con un número par de bits, la paridad no es correcta. Esto podría significar que uno de los bits de caracteres se perdió durante la transmisión. Cuando no se logra la paridad, esto indica un error en la transmisión. En este caso, el software de comunicación le indica que intente nuevamente la transmisión de datos o se reinicia automáticamente.
