Running the Program. To load and run the System Diagnostic Program, follow these steps: Turn power on to the system. Applesoft BASIC is loaded automatically, and the ] prompt is displayed. Place the diagnostic diskette into the disk drive. Enter RUN DIAG. If you have two disk drives and you're loading the program from drive#2, enter RUN DIAG,D2. A beep indicates that the loading process has begun. A second beep indicates that the System Diagnostic Program has completed the load. The System Diagnostic Main Menu is then displayed. Follow the same steps to run the High-Resolution Graphics Color Test. Enter `RUN HIRES`. The High-Resolution Graphics Color Test instructions will be displayed.
